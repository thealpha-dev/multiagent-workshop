Hands-on: Environment Setup
1. Python Environment Setup

Ensure Python 3.8+ is installed, then create a virtual environment:

# Create virtual environment
python3 -m venv venv_mas

# Activate on macOS/Linux
source venv_mas/bin/activate

# Activate on Windows
.\venv_mas\Scripts\activate
2. Install LangChain and Dependencies

Install the required packages:

# Install LangChain
pip install langchain

# Install OpenAI (or other LLM provider)
pip install openai

# Install additional tools
pip install wikipedia
3. Set Up API Keys

Set your OpenAI API key as an environment variable:

# On macOS/Linux
export OPENAI_API_KEY="your-api-key-here"

# On Windows
set OPENAI_API_KEY=your-api-key-here

This environment will serve as the foundation for building multi-agent systems using various frameworks.
