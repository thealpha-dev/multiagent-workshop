Overview of Frameworks
LangChain

An open-source framework that simplifies the creation of applications using large language models (LLMs) by connecting them with external data sources.

LangGraph

A stateful orchestration framework that structures tasks as nodes and transitions as edges, providing a clear workflow architecture.

AutoGen

Microsoft's framework for building AI agents and facilitating cooperation among multiple agents through conversation-driven collaboration.

CrewAI & n8n

CrewAI: A lean Python framework for creating collaborative, autonomous AI agents.
n8n: A workflow automation tool that connects applications and automates tasks.
